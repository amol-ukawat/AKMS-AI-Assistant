# AKMS AI Assistant

AKMS is a Python-based AI Voice Assistant built using CustomTkinter.

## Features

- Voice Commands
- Wake Word Detection
- Memory System
- Notes System
- PDF Reader
- Battery Status
- WiFi Status
- Screenshot
- Open Chrome
- Open VS Code
- Open WhatsApp

## Installation

pip install -r requirements.txt

python ui.py
